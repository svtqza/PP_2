# TSIS 2 Paint Application — Extended Drawing Tools

## Files

```text
TSIS2/
├── paint.py
├── tools.py
└── assets/
```

## Features

- Pencil / freehand drawing
- Straight line with live preview
- Rectangle
- Circle
- Square
- Right triangle
- Equilateral triangle
- Rhombus
- Eraser
- Color picker
- Brush sizes: small, medium, large
- Flood fill tool
- Text tool
- Ctrl+S saves canvas as PNG with timestamp

## How to run

Go to the folder:

```bash
cd /Users/zanajym/Desktop/PP2/TSIS2
```

Run:

```bash
python3 paint.py
```

## Keyboard shortcuts

```text
P - Pencil
L - Straight line
R - Rectangle
O - Circle
S - Square
T - Right triangle
Q - Equilateral triangle
H - Rhombus
F - Flood fill
X - Text tool
E - Eraser
C - Clear canvas

1 - Small brush, 2 px
2 - Medium brush, 5 px
3 - Large brush, 10 px

Ctrl+S - Save canvas
Enter - Confirm text
Escape - Cancel text
```
