# TSIS 3 Racer Game — Advanced Driving, Leaderboard & Power-Ups

## Files

```text
TSIS3/
├── main.py
├── racer.py
├── ui.py
├── persistence.py
├── settings.json
├── leaderboard.json
└── assets/
```

## Features

- Main Menu: Play, Leaderboard, Settings, Quit
- Username entry before game
- Lane hazards and safe paths
- Dynamic road events
- Traffic cars and random obstacles
- Safe spawn logic
- Difficulty scaling
- Weighted coins
- Enemy/traffic speed scaling
- Power-ups:
  - Nitro
  - Shield
  - Repair
- Score from coins and distance
- Distance meter and remaining distance
- Persistent leaderboard saved in `leaderboard.json`
- Top 10 leaderboard screen
- Settings saved in `settings.json`
- Sound toggle
- Car color selection
- Difficulty selection
